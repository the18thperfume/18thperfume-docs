# Risk Summary YAML Block cho Gate File
# Copy nội dung dưới đây vào phần risk_summary của gate file

```yaml
risk_summary:
  totals:
    critical: 2  # score 9
    high: 3      # score 6
    medium: 4    # score 4
    low: 3       # score 2-3
  highest:
    id: SEC-001
    score: 9
    title: 'Admin authentication bypass vulnerability'
  recommendations:
    must_fix:
      - 'Implement proper admin authentication với AWS Cognito'
      - 'Add data integrity mechanisms với DynamoDB transactions'
      - 'Comprehensive input validation cho tất cả admin forms'
    monitor:
      - 'Failed authentication attempts rate'
      - 'Admin session anomalies và unauthorized access'
      - 'Database query performance cho admin operations'
      - 'Data validation failure rates'
```
